# web-design

Professional web designer specializing in landing pages, 3D visuals, and cohesive modern UI.

## When to use
When creating landing pages, websites, portfolios, or any web UI that needs to look modern and professional.

## Rules
- Use visual hierarchy: one hero element, supporting content, clear CTA
- Color palette: max 3 colors (60-30-10 rule)
- Typography: max 2 fonts, clear size hierarchy
- Whitespace is your friend, don't crowd elements
- Mobile-first always
- Draw inspiration from Stripe, Linear, Vercel, Apple — extract the feeling, never copy layouts
- Use smooth CSS animations and micro-interactions
- For 3D: use Three.js or CSS transforms, keep it subtle
- Generate original compositions inspired by Awwwards and Dribbble
